## FAQ: Therapy Billing Code Software

### 1. **What are the most common therapy billing codes in the US?**  
In the US, therapists most frequently use **CPT code 90834** (individual psychotherapy) and **90837** (group therapy). Medical therapists also use **90832** for 15-minute evaluations. For ICD-10, "F43.12" (adjustment disorder with anxiety) is common, but coding depends on diagnosis documentation.

### 2. **How does SimplePractice streamline billing code management?**  
SimplePractice allows you to create **custom billing templates** to map session types to CPT codes (e.g., "Initial Session" = 90832). While it doesn’t track ICD-10 codes automatically, therapists can manually add diagnosis codes to each session. Automated claims reduce manual data entry for routine billing.

### 3. **What are TherapyNotes’ main advantages for billing compliance?**  
TherapyNotes includes **preloaded ICD-10 and CPT code databases** that update automatically. It also validates codes during claim submission to prevent errors. The audit trail function lets you track changes to codes over time, which is useful for compliance audits.

### 4. **Which software handles out-of-network billing better?**  
SimplePractice is **better for out-of-network scenarios** because it lets you generate custom statements with codes and fees tailored to each client’s payment terms. TherapyNotes focuses on in-network billing and uses code libraries that may complicate out-of-network claims.

### 5. **Can I use both tools for different billing needs?**  
If you work with **mixed patient panels** (in-network and out-of-network), consider using TherapyNotes for in-network claims (for its code libraries) and SimplePractice for out-of-network billing. Both tools offer **HIPAA-compliant data export**, so you can sync financial records manually between platforms.
